# Comandos do POSTGRES

(ctrl + L) -> Limpa o terminal

Listar todos os dbs

```
\l
```

Listar todos os usuarios criados

```
\du
```

Conectando em um banco dentro da shell

```
\c m9_t22
```

Listando tabelas do banco

```
\dt
```

Descrever estrutura de uma tabela

```
\d nome_da_tabela
```
